## Study and Optimize the D3-Force Network Layout
### 命令说明
```
 npm run pretest 修改src代码，生成build/d3-force.js
 npm run test 运行测试代码
 npm run prepare d3-force库发布之前的准备，生成压缩之后的d3-force.min.js
```
### 演示测试Demo相关依赖
```
    "d3-collection" // Handy data structures for elements keyed by string.
    "d3-dispatch"  // Register named callbacks and call them with arguments.
    "d3-quadtree" // Two-dimensional recursive spatial subdivision.
    "d3-timer" // An efficient queue for managing thousands of concurrent animations.
    "d3-request" // A minimal, promise-based implementation to require asynchronous module definitions.
    "d3-selection" // Transform the DOM by selecting elements and joining to data.
```
